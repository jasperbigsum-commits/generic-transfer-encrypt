Transfer Encrypt End-to-End Demo

当前页面由 demo 服务直接提供，接口请求走仓库统一的浏览器传输加密协议。页面会先拉取服务端临时生成的 SM2 公钥，再自动完成 JSON、form、query、上传和下载的联调。

JSON

验证 JSON 请求解密和 JSON 响应加密。

Form

验证 `layui form bridge` 提交时，自动走 `application/x-www-form-urlencoded` 加密请求。

Query

验证 GET query 整体加密后再由服务端恢复。

Upload

验证文件上传不加密，但自动追加 `__md5_` 字段。

Upload Multi

验证同字段多文件时使用 `__md5_field__index` 规则。

Download

验证二进制下载不加密，但会校验 `X-Transfer-Content-MD5`。

OpenFeign

验证浏览器入站加密 + 服务端 OpenFeign 出站加密 + 下游再次解密的完整链路。
