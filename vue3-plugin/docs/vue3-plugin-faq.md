# Vue 3 Plugin FAQ

## 1. 为什么不直接基于 axios？

这里先做的是协议层和 Vue 3 注入层，不强制项目切换网络库。默认使用浏览器 `fetch`，好处是依赖更少、协议更直接。

## 2. 这个插件会替代 HTTPS 吗？

不会。这是应用层加密，HTTPS 仍然必须保留。

## 3. 为什么上传文件不加密？

与仓库服务端规则保持一致。当前默认只做 MD5 完整性校验，不对文件内容做二次加密。

## 4. 可以在非浏览器环境用吗？

可以，但前提是你自己提供：

- `fetchImpl`
- `cryptoImpl.getRandomValues`
- `FormData / File / Blob` 或兼容实现

## 5. 为什么还需要手动注入 smCrypto？

因为仓库目标是内网离线可用，不把第三方国密实现和分发方式强绑定在插件里。
